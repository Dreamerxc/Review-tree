#### 二叉树系列问题

##### 一，二叉树最大直径  543

```
int res;
    int diameterOfBinaryTree(TreeNode* root) {
        res = 0;
        depth(root);
        return res;
    }

    int depth(TreeNode* root) {
        if (root == nullptr) return 0;
        int left = depth(root->left);
        int right = depth(root->right);
        res = max(res, left + right);
        return max(left, right) + 1;
    }
```
