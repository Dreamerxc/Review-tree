#### 不同的搜索树

给你一个整数 `n` ，求恰由 `n` 个节点组成且节点值从 `1` 到 `n` 互不相同的 **二叉搜索树** 有多少种？返回满足题意的二叉搜索树的种数。

![img](https://assets.leetcode.com/uploads/2021/01/18/uniquebstn3.jpg)



##### 方法一：动态规划

```
int numTrees(int n) {
        vector<int> dp(n+1);
        dp[0] = 1;
        dp[1] = 1;
        for (int i = 2;i<=n;i++) {
            int val = 0;
            for (int j = 1;j<=i;j++) {
                val += dp[j-1] * dp[i-j];
            }
            dp[i] = val;
        }
        return dp[n];
    }
```
