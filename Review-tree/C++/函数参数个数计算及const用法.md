##### 一，函数参数个数计算

```
func(rec1,rec2+rec3,(rec4,rec5));
```

上面这个函数实在参数个数为3个

实参1=rec1

实参2=rec2+rec3

实参3=(rec4,rec5)

"实参"是在调用时传递给函数的参数. 实参可以是**常量、变量、表达式、函数**等， 无论实参是何种类型的量，在进行函数调用时，它们都必须具有确定的值。

##### 二，const语法问题

X为C++类

```
const X const *x;
```

用法错误，const重复使用了

```
X *const x;
```

const在*右侧，x指针不能够修改，但是x指针没有进行初始化，运行发生错误。
